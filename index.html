Help & Instructions

How to use this translator:

  1. Select translation direction (Text to Morse or Morse to Text)
  2. Type or paste your text in the input area
  3. Click "Translate" to convert your text
  4. For Morse to Text, separate letters with spaces and words with "/"
  5. Use the playback controls to hear your Morse code

Features:

  • Dark/Light Mode: Toggle between color themes
  • Speed Control: Adjust Morse code playback speed
  • Translation History: View your past translations
  • Copy Function: Copy results to clipboard
  • Sample Texts: Load example texts to try
  • Reference Chart: Built-in Morse code cheat sheet

Tips:

  • Playback works best with headphones
  • For long texts, consider breaking into smaller chunks
  • Letters are case-insensitive in text-to-Morse translation
